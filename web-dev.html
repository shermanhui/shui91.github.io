Web Dev Projects

Calorie Tracker

My first Single Page Application created using Backbone's MV* Framework coupled with Firebase for persistent storage. I took advantage of Backbone's flexibility and created an application with decoupled Views and used an Event bus to coordinate DOM manipulations with View refreshes such as changes in the Router's URL. I also familiarized myself with UnderscoreJS using its templating system and built in functions.

Neighbourhood Map

This was my very first project involving an MV* Framework. KnockoutJS was a great introduction to the concept of separation of concerns. I also extensively dove into JavaScript promises while learning how to work with multiple APIs (Google Maps, Instagram, FourSquare). This project also includes a Gulp build process and uses a HTML5 Boilerplate template designed with responsiveness in mind.

Asteroids Clone

I took Udacity's Frogger game clone to the next level. I implemented randomized asteroids and the "physics" invovled with the spinning asteroids wrapping around the screen and laser collisions. This project emphasized on a lot of Object Orientated JavaScript pratices, such as creating Laser, Asteroid and Ship classes, as well as Groups of each class to manage collision mechanics. This HTML5 asteroid game was inspired by Rice University's Coursera project "RiceRocks".

Frogger Replica

Implemented JavaScript's Object Orientated pseudo-classical style to replicate the classic Frogger arcade game. Game art and engine were provided by Udacity.
